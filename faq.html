KERN - FAQ

NOTE

Kern is in it early stages, and instructions/answers here may change over time.

What is KERN?

KERN is a bi-annually release set of radio astronomical binary software packages. It should contain most of the required tools that a radio astronomer require for setting up a data reduction pipeline.

KERN is just a set of packages, but maybe in the future derivates like container (docker) images will be created.

What is the audience for KERN?

KERN is intended to be used by scientists who want to work with Radio Astronomical data.

How does the release schema looks like?

We will have a cutting edge repository that will be updated with the latest packages. Every half year we will freeze this repository and do a KERN release. This release will then only contain critical bug fixes but not functional changes. This all to guarantee scientific reproducibility.

How do I install KERN?

For now there is only a development version which you can use.

What about docker images?

The Ubuntu packages can easily be used to create (custom) docker images. Base docker images will be supplied.

How do I request an update or the inclusion of a package in KERN?

Open up a bug report in our packaging issue tracker. Note that we don't add packages to an existing release. Also an existing KERN release is only updated when there is a dramatically serious problem.

Why did the first KERN release get version number 0?

The predecessor of KERN is the radio-astro PPA hosted at launchpad: https://launchpad.net/~radio-astro/+archive/ubuntu/main

We name this repo KERN-0 and leave it as it is now.

What are the requirements?

For now we base KERN on the latest Ubuntu LTS with a delay of 6 months up to a year. For KERN-1 this is Ubuntu 14.04, KERN-2 will be based on Ubuntu 16.04. We might switch the package base to something more flexible in the future though.

Can I mix KERN releases?

No, This is a really bad idea.

When is the next KERN version released?

There is no fixed release date, but we intent to release a KERN release every 6 months.
