Older releases

The old releases are KERN-0 for Ubuntu 14.04, KERN-1, KERN-2, KERN-3 for Ubuntu 16.04. KERN-4, KERN-5 and KERN-6 for Ubuntu 18.04. These releases have reached end of live and are not supported anymore. You can still use them though, also inside Docker containers.
